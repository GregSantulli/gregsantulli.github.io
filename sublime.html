License Sublime Text 2 for free on OSX

Enter the following commands into your terminal. Commands 2, 3 and 4 will print a lot of text to the screen, but just keep entering them exactly as listed:

  1. Navigate to the following directory:

cd /Applications/Sublime\ Text\ 2.app/Contents/MacOS/

  1. Open the file in vim

vim Sublime\ Text\ 2

  1. Change to hex mode:

:$!xxd

  1. Find and replace some type of magical code with another one ¯\_(ツ)_/¯ :

:%s/5BE509C33B020111/5BE509C32B020111/g

  1. Save and exit vim:

:x
