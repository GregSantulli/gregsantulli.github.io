T4: ENUMERABLE METHODS

The Enumerable module provides a set of methods to traverse, search, sort and manipulate collections. One of these methods is the cycle method. The cycle method iterates through a collection a given number of times:

cycle(n) { |obj| block }

Cycle will iterate the given block n times. If no argument is given, cycle will loop infinitely. See the below example:

array = ["a", "b", "c"]
array.cycle { |x| puts x }
# print, a, b, c, a, b, c,.. forever.
array.cycle(2) { |x| puts x }
# print, a, b, c, a, b, c.
